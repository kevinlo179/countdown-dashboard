# Countdown Dashboard

A production-ready first skeleton for a long-term countdown dashboard.

## Features in v1

- Today dashboard
- Event cards with countdown
- Custom icon and color
- Category, search, and status filter
- Drag-and-drop ordering
- Google Calendar quick-add link
- LocalStorage persistence
- Responsive design
- Ready for Supabase and Google Calendar API expansion

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This project includes `.github/workflows/deploy.yml` for automatic deployment.
