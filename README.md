# Countdown Dashboard

A static GitHub Pages countdown dashboard.

## Deploy

1. Upload these files to the root of your GitHub repository:
   - `.github/workflows/deploy.yml`
   - `index.html`
   - `README.md`
2. Go to Settings → Pages.
3. Select Source: GitHub Actions.
4. Push or commit changes.
5. Check the Actions tab for deployment status.

No npm, Node.js, React, or build step is required for this static version.
